## Version 

node - v24.7.0
pnpm - 10.15.0

## How to setup

Install nvm
nvm install 24
nvm use 24
npm install -g pnpm@latest-10

## developer setup

install eslint in vscode extensions

## references

## Command to run 

pnpm --filter [project-name] [command]

for example -

pnpm --filter warehouse-app dev
pnpm --filter admin dev
pnpm --filter warehouse-server dev
